[Mac OS] ignore accidental trackpad

Jul 7, 2017

I found that my cursor often jump exatic way when I type.

There's a solution.

System Preferences > Universal Access > Mouse & Trackpad tab > Trackpad options.. button > Check "Ignore built in trackpad when mouse or wireless trackpad is present"
