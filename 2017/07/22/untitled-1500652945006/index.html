[Hexo] SSL and Own Domain With Cloudflare

Jul 22, 2017

I have a blog as you see now on github-page builded by Hexo.

And I have an own domain "c1495616.com" bought from godaddy.

And I use Cloudflare as my DNS manager.

  1. add CNAME in source directory. (must put in source directory, otherwise disappear when you deploy. and add your wanted domain, ex : jerry.c1495616.com in CNAME.

  2. Cloudflare and add CNAME

  3. _config.yml change root to "/" instead of github directory name

  4. put the following in _layout.swig ( I use "next" as my template)

  5. if use custom js file in _layout.swig change the url (ex: /Blog/xxxx to //jerry.c1495616.com/XXX

  6. Cloudflare part: i) check the cloud, let it become orange. ii) Add page rule: ex: jerry.c1495616.com => type: jerry.c1495616.com/* And choose SSL to become "flexible"

P.S. You can check whether your SSL is valid or not by this website.
